## v1-RAG-Chatbot-Watsonx (2025)

The original project was created as a final project for **IBM’s _Generative AI Applications with RAG and LangChain_** course.

It is a Retrieval-Augmented Generation (RAG) chatbot that allows you to:

- Upload any PDF file
- Ask questions about its contents
- Receive natural language answers generated by **Watsonx's Mixtral-8x7B** LLM, grounded in retrieved document chunks

Initial prototype was built with:

- **IBM Watsonx Mixtral** model
- PDF document QA
- **LangChain** retrieval
- **Chroma** vector database
- **Gradio** interface
